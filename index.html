                <h2> How it works </h2>
                <p>
                    Based on the width (or height) of a container and the width (height) of a cell unit, It will create a virtual matrix. Scaning the matrix at each cell will find a free cell around to made a free area, then try to fit a block element into it. In case no block can fit the gap, it will resize the block to fill the gap (that is one of the options).
                </p>

            <div class='cover'>
                <div class='float-left'>
                    <h2>Options</h2>
                    <ul>
                        <li> animate </li>
                        <li> block </li>
                        <li> cell </li>
                        <li> fillGap </li>
                        <li> gutter </li>
                        <li> selector </li>
                    </ul>
                </div>
                <div class='desc'>
                    <h2>
                        <span>Options</span>
                        <a href="javascript:void(0);" class="float-right back-icon"></a>
                    </h2>
                    <h3>All options are parameters of Reset method</h3>
                    <ul>
                        <li>
                            <h3> animate : boolean </h3>
                            <div> True to make block move with animation</div>
                        </li>
                        <li>
                            <h3> block.flex : boolean </h3>
                            <div> True to made block can resize a bit to fit</div>
                            <div> False of block has class 'fixed' to keep size of block can't change</div>
                        </li>
                        <li>
                            <h3> cell.width : number </h3>
                            <div> The width of unit, base on it will build grid container.</div>
                        </li>
                        <li>
                            <h3> cell.height : number </h3>
                            <div> The height of unit, base on it will build grid container.</div>
                        </li>
                        <li>
                            <h3> fillGap : boolean </h3>
                            <div> True will made layout without gap.</div>
                        </li>
                        <li>
                            <h3> gutter : mixed </h3>
                            <div> The spacing between the blocks. Default is number, but can set it with 'auto' value.</div>
                        </li>
                        <li>
                            <h3> selector : string </h3>
                            <div> Get all blocks to made layout</div>
                        </li>
                        <li>
                            <h3> Example </h3>
                            <pre>
                            <code>
                                var wall = new freewall('.free-wall');
                                wall.reset({
                                  selector: '.brick',
                                  animate: true,
                                  block: {
                                    flex: true
                                  },
                                  cell: {
                                    width: 150,
                                    height: 150
                                  },
                                  fillGap: false
                                });
                            </code>
                            </pre>
                        </li>   
                    </ul>
                </div>
            </div>
        </div>

        <div class="brick size22 methods">
            <div class='cover'>
                <div class='float-left'>
                    <h2>Methods</h2>
                    <ul>
                        <li> fitHeight </li>
                        <li> fitWidth </li>
                        <li> fitZone </li>
                        <li> fixSize </li>
                        <li> reset </li>
                        <li> setFilter </li>
                        <li> unsetFilter </li>
                    </ul>
                </div>
                <div class='desc'>
                    <h2>
                        <span>Methods</span>
                        <a href="javascript:void(0);" class="float-right back-icon"></a>
                    </h2>
                    <ul>
                        <li>
                            <h3> fitHeight(height) </h3>
                            <div> Made the layout fit with limit height</div>
                        </li>
                        <li>
                            <h3> fitWidth(width) </h3>
                            <div> Made the layout fit with limit width</div>
                        </li>
                        <li>
                            <h3> fitZone(width, height) </h3>
                            <div> Made the layout fit with limit width and height</div>
                        </li>
                        <li>
                            <h3> fixSize(object) </h3>
                            <div> Set a block with special width or height</div>
                            <h5> Example </h5>
                            <pre>
                            <code>
                                var wall = new freewall('.free-wall');
                                var dna = $(".free-wall .flex-layout");
                                wall.unsetFilter();
                                wall.fixSize({
                                    block: dna,
                                    width: 150,
                                    height: 150
                                });
                                wall.fitWidth();
                            </code>
                            </pre>
                        </li>
                        <li>
                            <h3> reset(options) </h3>
                            <div> Config options for render layout.</div>
                            <h5> Example </h5>
                            <pre>
                            <code>
                                var ewall = new freewall(demo);
                                var demo = $(".example");
                                ewall.reset({
                                  selector: '.cell',
                                  animate: false,
                                  block: {
                                    flex: 1
                                  },
                                  cell: {
                                    width: 150,
                                    height: 150
                                  },
                                  fillGap: true
                                });
                                ewall.fitWidth(cwidth);
                            </code>
                            </pre>
                        </li>
                        <li>
                            <h3> setFilter(selector) </h3>
                            <div> Fillter blocks to show</div>
                        </li>
                        <li>
                            <h3> unsetFilter() </h3>
                            <div> Made all block to show</div>
                        </li>
                    </ul>
                </div>
            </div>
        </div>
